**🍕 Pizza Sales Analysis (SQL Server)**



--------------------------------------------------------------------------------------------




**📌 Project Overview**

This project focuses on analyzing pizza sales data using SQL Server to uncover valuable business insights. The dataset consists of multiple relational tables containing information about orders, order details, pizzas, and pizza types. By joining these four tables, SQL queries are used to analyze sales trends, customer ordering patterns, and product performance.

The analysis answers key business questions such as determining the total number of orders placed, calculating total revenue from pizza sales, identifying the highest-priced pizza, and finding the most commonly ordered pizza size. It also highlights the top five most ordered pizza types, helping identify the most popular menu items among customers.

---------------------------------------------------------------------------------------------


**📊 Analytical Insights**

The project further explores deeper business insights by analyzing:

* Category-wise pizza demand

* Distribution of orders by hour of the day

* Average number of pizzas ordered per day

* Top pizza types based on revenue

* Revenue contribution of each pizza type

* Cumulative revenue growth over time

* Top-performing pizzas within each category

These analyses help understand customer preferences, product popularity, and revenue trends, which can support better business decisions.
